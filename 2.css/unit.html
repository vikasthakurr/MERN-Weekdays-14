Understanding CSS Units

1. Absolute Units

Absolute units are fixed and will appear as exactly that size.

Pixels (300px)

2. Relative Units

Relative units specify a length relative to another length property.

Percentage (%): Relative to parent width.

100% of Container Width

EM vs REM:

Parent div has font-size: 20px
This is 2em (40px)
This is 2rem (32px - relative to root 16px)

3. Viewport Units

Relative to the browser window (viewport) size.

25vw (Width)

This page's container padding is also using rem, and your current screen width is used to calculate the vw box above.
