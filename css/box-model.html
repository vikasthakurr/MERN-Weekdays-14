The CSS Box Model

In CSS, the term "box model" is used when talking about design and layout. It is essentially a box that wraps around every HTML element.

CONTENT AREA
Content
Padding (Hidden inner space)
Border
Margin (Outer space)

Practical Tip: box-sizing: border-box;

By default, the width of an element only applies to the content. Adding padding and borders makes the element wider than you intended. Using border-box makes the width include padding and borders, which is much easier to manage!
